ALL REQUIRED SCRIPTS FOR THIS WEBSITE ARE IN ./project-master/package.json

### QUICK START GUIDE ###

1. NPM and Node required to run this website. No setup guide included here because Miro is lazy. This should be quite easy find online
2. Navigate to project-master folder in terminal

### RUN THE FOLLOWING IN TERMINAL ###

3. "npm run install-everything" to install the node_modules required to run this website
4. "npm run dev" to start the server and client

The website should automatically start in browser after running these commands. Details regarding known issues are listed in the Teams chat
